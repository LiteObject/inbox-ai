Google Calendar

Connect your Google Calendar to sync follow-up tasks as calendar events.

hourglass_empty Loading calendar status...
info

Setup Required: Before connecting, add your Google OAuth credentials:

  1. Go to Google Cloud Console
  2. Create/select a project and enable the Google Calendar API
  3. Create OAuth 2.0 credentials (Web application type)
  4. Add http://localhost:8000/calendar/callback to authorized redirect URIs
  5. Copy the Client ID and Client Secret to the form above
  6. Set INBOX_AI_CALENDAR__ENABLED=true in your .env
  7. Save settings and restart the server
